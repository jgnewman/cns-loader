# Cream & Sugar Loader for Webpack (Alpha)

## Usage

[Documentation: Using loaders](http://webpack.github.io/docs/using-loaders.html)

### Recommended configuration

``` javascript
{
  module: {
    loaders: [
      { test: /\.cns$/, loader: "cns-loader" },
    ]
  }
}
```
